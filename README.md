# Deepwater ROV Autonomous Manipulation System

## Overview
Developed a Python-based simulation of a deepwater ROV capable of manual control and autonomous object retrieval.

## Features
- Autonomous navigation and object pickup
- Manual control mode
- Multi-object handling
- Battery and depth constraints
- State machine (SEARCH → NAVIGATE → PICK → RETURN)

## Tech Stack
- Python

## How to Run
```bash
python rov_autonomous.py
